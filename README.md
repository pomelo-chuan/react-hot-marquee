# Installation

## With npm

```bash
$ npm install --save react-hot-marquee
```

## With yarn

```bash
$ yarn add react-hot-marquee
```

# Usage

## Simply usage

```javascript
import ReactHotMarquee from "react-hot-marquee";

function Demo() {
  return (
    <>
      <ReactHotMarquee>
        Yesterday, All my trouble seemed so far away.
      </ReactHotMarquee>
    </>
  );
}
```
